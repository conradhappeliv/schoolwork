ImageMorpher

About

At the simplest level, Image morphing is simply the blending of pixels of image I and J to create M using the equation

M(x,y) = (1-alpha)I(x,y) + alpha*J(x,y)

0 ≤ alpha ≤ 1

Simple intensity morphing does not account for misaligned faces, so M will usually result in a blur of I and J. We need to establish pixel correspondence between I and J. Using dlib, we detect each facial feature point in both images. #potentially insert an image example. We use that set of detected feature points, as well as corners of the image to generate a list of Delaunay Triangles. #potentially insert an image example. The graphs formed by the Delaunay Triangulation on two faces are isomorphic. That means that every point or edge of one graph/image can be mapped to another point or edge of the second graph. Consequently, each triangle generated by a subset of 3 points, can be mapped to a triangle in the other image using the same mapped points. Additionally, a triangle formed from the points generated by a linear combination of two triangles, is also isomorphic. We use OpenCV's getAffineTransform function to calculate the transforms needed to get triangles from image I and J affined to the combined triangles of image M. We then use OpenCV's warpAffine function on the bounding boxes for those triangles and combine the result with a triangular mask using OpenCV's fillConvexPoly.
